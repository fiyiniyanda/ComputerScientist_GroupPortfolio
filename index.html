📋 how to replicate & contribute (team workflow)

  • ➕ Add a pet for a friend: copy a <div class="pet-card">...</div> block, paste inside the .pets-collection of your teammate's section, then change name, type, color and emoji/image.
  • ➕ Add a new interest: copy <li>...</li> inside the .interests-list of any person.
  • ➕ Create an entirely new section: copy a whole <div class="extra-creative">...</div> and paste it before the closing </section> of a person card. Customize the heading and text to reflect a new fact (e.g., "favorite games", "dream travel").
  • ➕ Add a new person (new group member): copy the whole <section class="person-card">...</section> from any existing member and paste before </main>. Fill in all sub-blocks (avatar, usernames, aspirations, interests, pets).

⭐ Pro tip: For a self-portrait image instead of animated emoji, replace the .avatar-icon div with <img class="avatar-img" src="your-image-url.jpg" alt="portrait">. Use any icon library or GIF for animated characters!

🌐 built with collaborative HTML blocks — each person can clone any section. Share your screen, paste new code, and fill with your teammate's details!
